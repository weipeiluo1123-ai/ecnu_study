# study

算法刷题 & 开发工具学习笔记。

## 目录结构

```
├── algorithm/          # 算法刷题（洛谷 / LeetCode）
│   ├── docs/           #   题目题解笔记（13 题）
│   ├── leetcode/       #   LeetCode 题解（2 题）
│   ├── images/         #   题解插图（20 张）
│   ├── include/        #   自定义头文件
│   ├── templates/      #   算法模板
│   └── notes/          #   学习笔记
├── toolbox/            # 开发工具学习
```

- LeetCode 题解（持续更新）

### toolbox — 工具链
- **Git**: 版本控制学习记录
- **Vim**: 编辑器操作笔记

### docs — 学习规划
