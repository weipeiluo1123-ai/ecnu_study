# study

算法刷题 & 开发工具 & 项目实战学习笔记。

## 目录结构

```
├── algorithm/              # 算法刷题（洛谷 / LeetCode）
│   ├── luogu/              #   洛谷题解笔记（13 题，含插图）
│   ├── leetcode/           #   LeetCode 题解（2 题）
│   ├── images/             #   题解插图（20 张）
│   ├── include/bits/       #   自定义头文件
│   ├── templates/          #   算法 / 题解模板
│   ├── Makefile            #   C++ 编译脚本
│   └── .vscode/            #   VSCode 调试配置
├── toolbox/                # 开发工具链学习笔记
│   ├── git/                #   Git 笔记（sparse-checkout 等）
│   ├── vim/                #   Vim 笔记
│   └── shell/              #   Shell 笔记（Missing Semester）
├── docs/                   # 方向规划 & 文档
├── toy/                    # 练手项目
│   ├── my-blog/            #   Next.js 全栈博客
│   └── my-ai-coder/        #   AI 编码工具
└── README.md
```

## 各模块说明

### algorithm — 算法题解

- **洛谷**（C++）：栈、队列、哈希表、二分、模拟、STL 等基础算法
- **LeetCode**（C++）：持续更新
- 每题含：题目分析、代码实现、运行截图 / 示意图

### toolbox — 工具链

- **Git**: 版本控制、sparse-checkout 等
- **Vim**: 编辑器操作笔记
- **Shell**: Missing Semester 系列笔记
