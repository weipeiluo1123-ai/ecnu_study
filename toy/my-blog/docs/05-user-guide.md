# 用户指南

## 角色说明

| 角色 | 默认权限 | 说明 |
|------|---------|------|
| 游客 | 浏览、搜索 | 不需要登录 |
| 普通用户 | 评论、点赞、收藏、发布文章、改名 | 注册即获得 |
| 管理员 | 全部权限 + 用户管理 + 审核管理 | 系统预设 |
| 超级管理员 | 全部权限 + 不可被修改/删除 | 系统预设，唯一 |

## 注册与登录

### 注册

1. 点击导航栏右侧的 **登录** 按钮
2. 选择 **注册** 标签
3. 填写用户名（2-20 字符）、**邮箱地址**
4. 点击 **发送验证码** 按钮
   - 系统会向填写的邮箱发送 6 位数字验证码
   - 验证码 5 分钟内有效
   - 60 秒后可重新发送
   - 如果邮箱不存在，系统会提示"该邮箱地址不存在"
5. 在邮箱查收验证码，填入输入框
6. 填写密码（至少 6 位）
7. 点击 **注册**，验证通过后自动登录并跳转首页

### 登录

1. 点击导航栏的 **登录**
2. 输入用户名和密码
3. 默认管理员：`admin / admin123`

### 退出

点击右上角用户头像 → **退出登录**

## 互动操作

### 评论

1. 打开任意文章
2. 滚动到底部的评论区
3. 登录后在输入框写内容，点击"发表评论"
4. 可以删除自己的评论（点击评论旁的垃圾桶图标）

### 点赞

**文章顶部信息栏** 和 **文章底部操作栏** 均有点赞按钮：
- 未登录：点击跳转登录页
- 已登录：点击切换点赞状态
- 底部按钮更大（椭圆形），更显眼

### 收藏

**文章顶部信息栏** 和 **文章底部操作栏** 均有收藏按钮：
- 未登录：点击跳转登录页
- 已登录：点击切换收藏状态
- 底部按钮更大（椭圆形），更显眼
- 收藏后可在 **我的收藏** (右上角头像 → 我的收藏) 中查看所有收藏的文章列表

### 文章卡片互动数据

在文章列表页（首页、分类页、标签页），每张文章卡片上直接显示：
- ❤️ 点赞数
- 🔖 收藏数
- 📅 发布日期
- 👁 浏览量（仅文章详情页显示）

> 不再显示"X 分钟阅读"时长。

### 浏览量

每篇文章自动统计浏览量，**每次访问 +1**（无去重限制）

## 用户主页

每个用户都有一个个人主页，可通过点击任意位置的头像或用户名访问：
- 文章评论区点击作者名
- 排行榜点击用户名
- 文章卡片点击作者名

用户主页展示：
- 用户头像、用户名、角色、加入时间
- 该用户发布的文章列表
- 该用户的评论列表

**超级管理员特殊标识：**
- 头像为金色皇冠图标，带有渐变背景和发光效果
- 角色标签为"超级管理员"（金色）
- 在页面各处（导航栏、文章卡片、评论区）均有专属标识

## 发布文章

### 方式一：管理员发布 MDX 文章

在 `content/posts/` 目录下创建 `.md` 文件：

```bash
# 1. 创建文件
touch content/posts/my-new-post.md

# 2. 写入内容（含 YAML 头部）
# ---
# title: "文章标题"
# date: 2026-05-01
# description: "摘要"
# category: "frontend"
# tags:
#   - react
# published: true
# ---
#
# ## 正文内容...

# 3. 重新构建
npm run build
```

### 方式二：用户通过网页发布

1. 登录后访问 `/posts/new`
2. 填写标题、选择分类、选择标签
3. 使用 Markdown 写内容
4. 点击发布
5. 发布成功后自动跳转回文章列表页，新文章即刻可见

> 文章列表页支持分页，每页可选择显示 5 / 10 / 20 篇。分类页、标签页、用户主页、我的文章、我的收藏均支持分页。

> ⚠️ 管理员可在后台控制用户是否有发布权限

## 管理后台（仅管理员）

### 访问

1. 用管理员账号登录（admin / admin123）
2. 点击右上角头像 → **管理后台**
3. 或直接访问 `/admin`

### 用户管理

1. 在后台点击 **用户管理** 或访问 `/admin/users`
2. 可以看到所有注册用户列表
3. 每个用户可以操作：

| 操作 | 说明 |
|------|------|
| 切换评论权限 | 控制用户能否评论 |
| 切换点赞权限 | 控制用户能否点赞 |
| 切换用户/管理员 | 提升或降级用户身份 |
| 删除用户 | ⚠️ 不可恢复（评论会匿名化保留，点赞/收藏会被清理） |

> 不能删除自己，不能修改自己的权限

### 审核管理

1. 在后台点击 **审核管理** 或访问 `/admin/reviews`
2. 有两个标签页：

**改名审核：**
- 展示所有用户改名记录（待审核 / 已批准 / 已驳回）
- 待审核的请求可操作：
  - ✅ **批准** — 保留新用户名
  - ❌ **驳回** — 回滚到原用户名
- 有未处理请求时标签上会显示数字角标

**文章管理：**
- 展示所有用户发布的文章
- 每篇文章可操作：
  - 👁 **上架/下架** — 控制文章是否对外可见
  - 🗑 **删除** — 永久移除文章（不可恢复）

### 评论管理

1. 在后台点击 **评论管理** 或访问 `/admin/comments`
2. 可以看到所有注册用户发表的全部评论（按时间倒序）
3. 每条评论显示：作者、内容、所属文章链接
4. 点击 **跳转到文章** 可以查看该评论的原文上下文
5. 管理员可以删除任意评论

> 删除用户账号后，其评论不会丢失，而是显示为"已注销"

## 个人设置

### 个人资料

1. 点击右上角用户头像 → **个人设置**，或直接访问 `/settings`
2. 展示以下信息：
   - 头像、用户名、邮箱、角色、加入时间
   - **总积分**、**月积分**、**周积分**、**文章数**（与排行榜数据关联）
3. 可编辑：
   - **个人简介** — 介绍自己、技术兴趣和爱好
   - **修改用户名** — 修改后立即生效，管理员可在后台审核

### 积分说明

| 指标 | 说明 | 关联 |
|------|------|------|
| 总积分 | 用户所有文章的累计积分 | 排行榜总积分 |
| 月积分 | 过去 30 天内获得的积分 | 排行榜月榜 |
| 周积分 | 过去 7 天内获得的积分 | 排行榜周榜 |

积分公式：`浏览量 × 1 + 点赞量 × 5 + 收藏量 × 10`

### 修改用户名

1. 在设置页的"修改用户名"区域输入新用户名（2-20 字符）
2. 点击保存，用户名立即更新
3. 管理员会在后台收到一条待审核记录
4. 如果管理员发现改名不合规，可以驳回并回滚

> ⚠️ 不能使用系统保留用户名（weipeiluo、admin、root、system 等）

## 文章管理

### 我的文章

1. 点击右上角用户头像 → **我的文章**，或直接访问 `/my-posts`
2. 展示当前用户发布的所有文章
3. 每篇文章可操作：
   - 👁 **查看文章** — 跳转到文章详情页
   - ✏️ **编辑** — 跳转到编辑页面修改文章
   - 🗑 **删除** — 永久删除文章（不可恢复）

### 编辑文章

在编辑页面可以修改：
- **标题** — 修改文章标题
- **分类** — 更改所属分类
- **标签** — 增删标签
- **摘要** — 修改文章简介
- **内容** — 使用 Markdown 编写，支持实时预览（切换 编辑/预览 模式）
- **发布状态** — 上架/下架文章

> 只能编辑自己发布的文章，管理员和超级管理员可以编辑所有文章

### 写文章

两种方式：

**方式一：管理员通过 MDX 文件**
在 `content/posts/` 目录下创建 `.md` 文件，需重新构建。

**方式二：所有用户通过网页**
1. 登录后访问 `/posts/new` 或在我的文章页点击"写文章"
2. 填写标题、选择分类、选择标签
3. 使用 Markdown 编写内容，支持实时预览
4. 点击发布
5. 发布成功后自动跳转到文章列表页

> ⚠️ 管理员可在后台控制用户是否有发布权限。超级管理员发布的文章在详情页会带专属皇冠标识。

## 搜索功能

### 文章搜索

1. 访问 `/search`，默认进入文章搜索
2. 输入关键词搜索文章标题、描述、标签和分类
3. 支持模糊匹配，自动高亮

### 用户搜索

1. 在搜索页切换至 **用户** 标签
2. 输入用户名或个人简介关键词
3. 结果以卡片形式展示，包含头像、角色标识和个人简介
4. 点击用户卡片跳转至其个人主页

## 排行榜

### 访问

导航栏 **排行** 或直接访问 `/leaderboard`

### 周期切换

四个标签页切换统计范围，默认展示 **总分**（全部时间累计）：

| 标签 | 统计范围 |
|------|---------|
| 总分 | 全部时间（累计） |
| 每日 | 过去 24 小时 |
| 每周 | 过去 7 天 |
| 每月 | 过去 30 天 |

### 积分规则

```
浏览量 × 1 + 点赞量 × 5 + 收藏量 × 10 = 文章积分
用户总积分 = 所有文章积分之和
```

### 自动刷新

排行榜数据每 **60 秒**自动刷新一次，切换周期标签时也会立即刷新。

## 常见问题

**Q: 忘记密码怎么办？**
A: 目前未实现密码找回功能，请联系管理员重置。

**Q: 评论可以编辑吗？**
A: 暂不支持编辑评论，可以删除后重新发表。

**Q: 为什么我发布的文章没有出现在排行榜？**
A: 排行榜统计的是用户文章的互动数据，新发布的文章需要获得浏览量/点赞/收藏才会累积积分。

**Q: 如何修改用户名？**
A: 访问 `/settings` 或在用户菜单中点击"个人设置"，输入新用户名即可。修改立即生效，同时管理员会收到审核通知，如果发现不合规可以撤回改名。

**Q: 改名被驳回了怎么办？**
A: 管理员驳回后用户名会自动回滚到原来的名称，你可以重新提交合规的改名申请。
