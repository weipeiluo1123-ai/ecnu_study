## 学习回顾

### React 生态

这个季度深入学习了 React 18/19 的新特性，特别是并发模式和 Server Components。

### Rust 入门

Rust 的所有权系统和 borrow checker 确实有陡峭的学习曲线，但掌握后能写出非常安全高效的代码。

### AI/LLM

LLM 应用开发是今年的重点方向，从 Prompt 工程到 RAG 架构，再到 Agent 开发，每周都有新的突破。

## 学习方法

### 输出驱动输入

通过写博客和做项目来驱动学习，效果比单纯看书好很多。

### 每日编码

保持每天至少 1 小时的编码时间，周末可以更多。
