# 【算法题】P1540 [NOIP 2010 提高组] 机器翻译
## 一、题目核心逻辑
核心实现思路：模拟 FIFO 页面置换算法，用`queue`记录，通过unordered_set配合实现查找。

## 踩坑点
### 核心容器函数差异
| 容器类型   | 代表容器 | 核心添加函数 | 是否支持 push_back | 核心说明  |
| --- | --- | --- | --- | --- |
| 队列（适配器） | queue | push() | ❌不支持 | 队列遵循先进先出规则，仅暴露 push ()（队尾入队）、pop ()（队首出队）接口，屏蔽底层实现细节 |
| 集合（关联式） | set/unordered_set | insert() | ❌不支持 | 集合核心是元素唯一性 + 快速查找，插入位置由哈希 / 排序规则决定，无 “尾部” 概念，仅用 insert () 插入 |
| 序列式容器 | vector/list/deque | push_back()/push_front() | ✅ 支持 | 线性存储结构，明确区分 “尾部 / 头部”，push_back () 语义为 “向容器尾部添加元素” |

## 最终AC代码
```cpp
#include <bits/stdc++.h>
using namespace std;

//P1540 [NOIP 2010 提高组] 机器翻译  -->模拟 FIFO 页面置换算法
int main(){
	int m, n;  //内存容量和文章的长度
	cin >> m >> n;
	
	queue<int> mem_q;
	unordered_set<int> mem_set;
	int count = 0;  //查词典次数 
	while(n--){
		int num;
		cin >> num;
		
		if(mem_set.find(num) == mem_set.end()){  //单词不在内存中
			count++;
			
			if(mem_q.size() >= m){  //替换
				int res = mem_q.front();
				mem_q.pop();
				mem_set.erase(res); 
			}
			mem_q.push(num);
			mem_set.insert(num);
		}
	}
	
	cout << count << '\n';	
	return 0;
}
```
