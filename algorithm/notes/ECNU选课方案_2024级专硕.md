## 二、选课策略

- 嵌入式方向，就业目标机器人/具身智能
- **春季尽量轻**，留出时间实习
- 选修课以好拿分为主，核心技能靠自学
- 信息技术创新方向需在「工业软件智能开发方法」和「区块链技术与商业应用」中二选一

## 四、关键判断

### 为什么学位基础课选「算法 + 软件理论」而非「计算机体系结构」

计算机体系结构固然重要（嵌入式基础），但它开在春季，选它意味着春季要多 3 学分、少一段实习时间。体系结构的内容可以自学（《计算机组成与设计：RISC-V版》），不一定要通过课程学。

### 为什么专业必修选「AI 基础 + 大数据」而非「嵌入式操作系统」

嵌入式操作系统开在春季，选它也会挤占实习时间。RTOS 的实操性极强，边做项目边学效率远高于听课，通过课题组项目和自学 FreeRTOS/Zephyr 补齐更高效。

### 为什么选修选「区块链」而非「工业软件」

两者对嵌入式方向都是"凑学分"，但区块链考核以案例分析+报告为主，好拿分且 3 学分能一次性凑够选修 7 分要求；工业软件偏技术实操、仅 2 学分，选它会多出一门选修缺口。

## 五、开学前准备建议

| 优先级 | 内容 | 目的 |
|:-----:|------|------|
| P0 | LeetCode 刷题（DP/贪心/回溯为主） | 直接对应算法课，减轻开学后压力 |
| P1 | C++ Primer 第 7~13 章 | 夯实 C++，写代码更快 |
| P2 | 读嵌入式方向顶会论文 1-2 篇/周 | 跟导师聊方向时已有定位 |
| P3 | Zotero / Overleaf 上手 | 开学即用，不临时抱佛脚 |
| - | 软件理论基础、区块链等课程 | 开学再学即可，不必提前焦虑 |
