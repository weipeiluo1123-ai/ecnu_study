## 一、核心刷题理念

### 1. 专题训练 → 螺旋上升

- **按知识点分类刷**而非随机刷：同一专题套路一致，效率最高
- **螺旋上升**：先完成低难度题（≤1700 分）打好基础，再挑战更难的题目
- 低难度题只考单一知识点，高难度题考多个知识点组合

### 2. 难度评分体系（Rating 0–4000）

LeetCode 周赛/双周赛每道题有难度分（rating），比 Easy/Medium/Hard 更精细：

| 你的水平 | 建议刷题分数段 |
|---------|--------------|
| 新手入门 | 先刷 **≤1700** 的基础题 |
| 有一定基础 | 做 **当前 rating + 200** 的题目 |
| 进阶提升 | 按 100 分为一段，逐段提升 |

> 推荐安装浏览器插件 **LeetCodeRating** 显示每道题的难度分。

## 二、三大刷题方法

### 方法 A：专题训练（核心 · 推荐）

按知识点系统刷题，适合**从零开始**的新手，也是本文档的重点。

### 方法 B：随机训练

用 LeetCode 的"难度练习"模式，**关闭算法标签**刷题，锻炼判断题目类型的能力（因为面试时没人告诉你考察什么算法）。

### 方法 C：突击训练（面试向）

时间紧迫时刷 **LeetCode HOT 100** 或 **面试 150** 题单。

## 三、从零开始的刷题路线（推荐顺序）

> 路线设计理念：先从**技巧性强、思路明确、刷起来快**的专题入手，建立信心，逐步过渡到**动态规划**等需要大量练习的核心专题。

```
T0（打基础）  →  技巧类（滑窗/双指针/二分/前缀和） → 递归理解（链表/树/回溯） → DP
T1（进阶）   →  图论、单调栈、位运算
T2（选学）   →  高级数据结构（并查集/树状数组/线段树）、数学、贪心
```

## 五、刷题建议与技巧

| 建议 | 说明 |
|------|------|
| **思考时间** | 思考 10 分钟没有思路 → 直接看题解（可能是某个概念不知道） |
| **看题解不丢人** | 不看题解硬磕才是效率最低的 |
| **举一反三** | 吃透有限题 > 盲目刷更多题 |
| **DP 题量门槛** | 至少做 100 道才算真正入门 |
| **记忆化搜索** | 先写记忆化搜索（自动挡）理解思路，再转递推（手动挡）|
| **每日打卡** | 保持节奏比单日刷很多更重要 |
